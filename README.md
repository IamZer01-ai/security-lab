# CyberSecurityA1 SOC v2

A professional Security Operations Center (SOC) themed cybersecurity portfolio and dashboard built using HTML, CSS, and JavaScript.

## Overview

CyberSecurityA1 SOC v2 is designed to simulate a modern Security Operations Center environment while showcasing cybersecurity skills, projects, investigations, threat intelligence, and security reporting capabilities.

The platform features a matrix-style cyber interface, interactive dashboard, real-time clock, security-focused navigation, and professional portfolio sections.
